# Drop videos here 🎬

Any video (mp4/webm) dropped in this folder appears in the reel automatically.

**How to add videos:** on GitHub, open this folder → **Add file → Upload files**
→ drag your videos in → **Commit changes**.

Note: GitHub's upload limit is **25MB per file**. Bigger videos: trim/compress
them first, or keep uploading them through Lovable chat as assets — those are
picked up automatically too.
