# Drop photos here 📸

Any image dropped in this folder appears on the site automatically —
sorted into the right month using the photo's own capture date (EXIF).

**How to add photos:** on GitHub, open this folder → **Add file → Upload files**
→ drag your photos in → **Commit changes**. That's it.

Tips:
- Use **original photos from your camera roll** — they keep their capture date.
  Photos saved from WhatsApp/Instagram lose it.
- No capture date? Put it in the filename: `2025-08-14_beach-day.jpg`.
- Captions and date fixes go in `src/lib/media-overrides.ts` (optional).
